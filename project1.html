This project is an implementation of a breadth-first search algorithm implemented in JavaScript programming language. This algorithm guarantees getting the shortest path from the starting node to the finishing node, in case there is a path. The memory consumption of this algorithm gorws proportionally to the distance between the start and finish. In terms of the big O notation, this algorithm takes approximately O(log(nÂ²)) time to finish, where n is the number of tiles between the starting position and the finish.

Apart from the pathfinding algorithm, this project includes a maze-generating algorithm based on recursive division and a zigzaging recursive algorithm added by me to make mazes look pretier.

You can change locations of the starting and finishing nodes by pressing buttons in the upper section of the web page, as well as use the above-described functions on the given board. You can create 'walls' by clicking anywhere on the grid. 'Clear board' clears everything that was created by you, 'clear path' deletes only the path nodes.

Have fun!
