This project is an implementation of a breadth-first search algorithm in the JavaScript programming language. The algorithm ensures finding the shortest path from the starting node to the finishing node, provided that a path exists. The memory consumption of this algorithm grows proportionally to the distance between the start and finish. In terms of the big O notation, this algorithm takes approximately O(nÂ²) time to finish, where n is the number of tiles between the starting position and the finish.

In addition to the pathfinding algorithm, this project features a maze-generating algorithm based on recursive division and a zigzagging recursive algorithm added by me to enhance the appearance of mazes.

You can change the locations of the starting and finishing nodes by pressing buttons in the upper section of the web page, as well as use the above-described functions on the given board. You can create 'walls' by clicking anywhere on the grid. 'Clear board' removes everything that you have created, while 'clear path' deletes only the path nodes.
